# aspire-todo-app

Aspire Todo App

A modern, clean, and fully functional Todo app built with Kotlin, Jetpack Compose, and Firebase. It follows a Clean Architecture approach with MVVM design pattern, SSOT, and UDF, making it scalable and easy to maintain.

Tech Stack
	•	Language: Kotlin
	•	UI: Jetpack Compose
	•	Architecture: MVVM, Clean Architecture
	•	Backend / Database: Firebase Firestore
	•	Dependency Injection: Hilt
  •	SSOT, UDF
	•	Coroutines & Flow: Kotlin Coroutines, StateFlow, SharedFlow
